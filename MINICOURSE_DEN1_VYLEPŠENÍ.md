## 📊 PŘED vs PO:

### **PŘED (4 kroky):**
```
1. Vyberte zákazníky
2. Pošlete email
3. Zaznamenejte odpovědi
4. Najděte vzorce

→ A TEĎ CO??? 🤷‍♂️
```

### **TEĎ (5 kroků):**
```
1. Vyberte zákazníky
2. Pošlete email
3. Zaznamenejte odpovědi (jejich slova!)
4. Najděte vzorce (3+ = skutečná hodnota)
5. CO S TÍM TEĎ → Kde použít (web, Instagram, atd.)

→ JASNÝ AKČNÍ PLÁN! ✅
```
