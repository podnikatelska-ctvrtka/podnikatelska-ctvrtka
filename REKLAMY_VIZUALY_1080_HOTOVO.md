## 💡 TIPS:

### **SCREENSHOT QUALITY:**
```
✅ USE:
• High resolution (1080x1080 native)
• PNG format (lossless)
• Full quality export
• Clean screenshot (no browser UI)

❌ AVOID:
• JPG (lossy compression)
• Low resolution
• Browser UI in screenshot
• Watermarks
```

### **FB UPLOAD:**
```
✅ BEST PRACTICE:
• Upload PNG directly
• Don't resize in FB
• Let FB optimize
• Test mobile preview

💡 TIP:
FB může komprimovat - check preview!
Pokud quality drop → upload znovu higher quality.
```

### **COPY MATCHING:**
```
✅ ENSURE:
• Visual + Text match
• Message alignment
• CTA consistent
• Offer clear

❌ DON'T:
• Mix visual from Ad #1 with copy from Ad #2
• Change messaging
• Confuse offers
```
